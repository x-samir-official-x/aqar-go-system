# Aqar-Go System 🏠

نظام ذكي متكامل لإدارة العقارات والخدمات، مصمم لتسهيل عمليات الإدخال والمتابعة للفريق.

## المميزات
- **لوحة تحكم (Dashboard):** متابعة إحصائيات العقارات والخدمات لحظة بلحظة.
- **إدارة الفريق:** إضافة وتعديل أدوار الأعضاء (Admin, Reviewer, Collector).
- **نظام المراجعة:** قبول أو رفض العقارات والخدمات قبل إضافتها للنظام.
- **التحليلات:** رسوم بيانية (Charts) توضح أداء الفريق وتوزيع العقارات.

## متطلبات التشغيل
- [Node.js](https://nodejs.org/) (v18+)
- [MongoDB](https://www.mongodb.com/) (Local or Atlas)
